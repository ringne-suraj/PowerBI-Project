# 📊 Motorola Sales Dashboard

## 📌 Overview
This Power BI dashboard analyzes mobile sales performance across cities, brands, and time.

## 📊 Key Metrics
- Total Sales: 769.20M
- Total Quantity: 19K
- Total Transactions: 4K
- Average Price: 40.11K

## 📈 Insights
- Sales peak in March & July
- Apple & Samsung lead sales
- UPI is most used payment method
- Weekend sales are highest

- Power BI
- Excel
